# Mega-Video

Web app for downloading videos from YouTube, Vimeo, and other platforms. Uses yt-dlp and ffmpeg.

```
git clone https://github.com/sarok-exe/Mega-Video.git
cd Mega-Video
bash run.sh
```

Open `http://localhost:8080` in your browser.
